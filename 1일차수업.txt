그 코드를 실행하게 해주는 프로그램

사용자정의명칭
 - 클래스명, 변수명, 메소드명
 - 영문대소문자, 숫자, $, _ 
 - 숫자는 첫글자로 올 수 없다
 - 특수문자는 $, _ 외에는 올 수 없다
 - 공백이 올 수 없다
 - 예약어는 사용 불가
자바개발자간의 약속
 - 클래스명의 첫글자는 영문대문자, $, _
   변수명, 메소드명의 첫글자는 영문소문자, $, _
 - 합성어의 첫글자는 반드시 영문 대문자로 

자바의 주석
 - // : 한줄주석
 - /* ~ */ : 여러줄주석

자료형, 변수, 상수
변수 : 변하는 수
상수 : 변하지 않는 수
자료형 : 변수나 상수등에 넣을 데이터 담을 그릇
 - 종류
   - 논리형  : boolean(1) - true, false
   - 정수형  : byte(1), -128 ~ 127
                 short(2), -32,768 ~ 32,767
                 char(2), 0 ~ 65535 
                 int(4), -2^31 ~ 2^31-1
                 long(8) -2^63 ~ 2^63-1
     - 기본자료형 : int
   - 실수형  : float(4), double(8)
      - 기본자료형 : double
      - float을 사용하려면 반드시 f나 F가 붙어야한다.
   - 문자열형 : String
 - 형변환 
   - 자동형변환 : 작은 자료형에서 큰 자료형으로 이동
   - 강제형변환 : 큰 자료형에서 작은 자료형으로 이동할때
                      앞에 (바꾸고자하는자료형)을 넣어준다
   - 실수형은 정수형보다 항상 크다
      double > float > long > int
   - char형이 문자형이라서 음수가 없음
   - int형이상의 자료형에서 연산결과는 큰 자료형값
   - int형이하의 자료형에서 연산결과는 int값

 - System 클래스
   - 입력 : in    System.in : 키보드로부터 값을 입력받는다
   - 출력 : out  System.out : 콘솔에 값을 출력하겠다
